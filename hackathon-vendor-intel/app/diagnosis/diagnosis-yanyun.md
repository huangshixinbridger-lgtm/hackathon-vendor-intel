## 燕云十六声 TikTok 生态分析

### 0. 口径说明
- 统计周期：近三个月，`2026-03-04` 至 `2026-06-02`。
- 模块范围：短视频 + 直播。
- 默认竞品/标杆：`Delta Force`、`Elden Ring`、`黑神话：悟空`、`逆水寒（Sword of Justice）`。这是在当前窗口内、两张主图表里都有稳定数据沉淀的 4 个对比对象。
- 短视频口径：`is_game_video = 1`，且 `p_date` 与 `item_create_day` 同窗，核心指标为总 VV、总投稿数、条均vv。[^1]
- 直播口径：沿用主图表 `groupby_type = 'game_name'`，核心指标为总看播时长、总看播人数、总开播主播、总开播时长、单位开播产出；国家结构因主图表无国家维度，使用同一数据集、同一时间窗和同一游戏筛选回查国家级汇总。[^2]

### 1. 执行摘要
燕云十六声当前更接近 **短视频驱动型** ，但短视频已经出现 **供给大于承接** 的问题，直播则处在 **有需求、但规模尚未进入第一梯队** 的阶段。[^1][^2]
建议 **继续投达人激励预算，但不宜粗放扩量** ；重点不是再把量堆上去，而是把短视频效率做上去、把直播的高承接国家补齐。[^1][^2]
预算优先投向 **短视频的高消费低供给国家** 和 **直播的高看播低开播国家** ，当前更值得优先看的市场是 **DE / VN / US / ID 的短视频** ，以及 **VN / US / ID 的直播** 。[^1][^2]

### 2. 核心判断
1. **总体标签** ：短视频驱动型；同时伴随达人数量大但内容效率低、国家供需错配两类典型诊断。[^1][^2]
2. **短视频判断** ：燕云短视频有规模、有供给，但消费效率没有进入第一梯队，属于“量先起来、效率没跟上”。[^1]
3. **直播判断** ：燕云直播不是没人看，而是主播规模和持续供给还没推到头部；单位开播产出已接近竞品中位，不是纯粹的低效盘。[^2]
4. **核心矛盾** ：短视频的问题在 **供给质量** ，直播的问题在 **供给规模与国家结构** 。

### 3. 短视频生态位分析

燕云的短视频已经具备大盘存在感，但当前更像 **供给先跑出来、消费承接仍落后于头部竞品** ；后续短视频预算应先提效，再扩量。[^1]

#### 3.6 短视频分析结论
短视频最清楚的信号是：
- **有供给，但供给结构不优** ：燕云总投稿数很高，但条均vv只有 1432，低于 Elden Ring 的 3757，也低于 Delta Force 的 3057。[^1]
- **国家错配明显** ：
  - **高消费低供给** ：VN、PK 的消费占比明显高于供给占比，DE 也更偏“消费拉得动”。[^1]
  - **高供给低承接** ：MY、FR、KR 的投稿占比明显高于消费占比，属于已经有人在发，但消费没有同步跟上的区域。[^1]
- **结论** ：燕云短视频现在不缺“发的人”，更缺“能把武侠开放世界、剧情、文化感做成高传播模板的人”。此前内部复盘也证明，燕云跨垂类内容合作、文化向/剧情向内容更容易形成外宣放大器，这一点值得继续放大。[^4]

#### 4.5 直播分析结论
直播侧的结论比短视频更明确：
- **问题不在单主播承接差，而在供给规模没补齐** ：燕云单位开播产出 2492，已经接近竞品中位，不属于“主播开了也没人看”的盘。[^2]

- **结论** ：直播侧仍可投，但应优先补 **能稳定拉看播时长的国家和主播层** 。历史活动复盘里，燕云在 US 首发节点通过直播活动拿到过非常强的看播爆发，但也暴露出水分问题，说明直播预算有杠杆，但更需要精细化定邀和质量控制。[^4]

   - **观察验证** ：PK（短视频消费高、供给低，但需结合业务目标小规模验证）

### 6. 预算建议
#### 6.1 该不该继续投
**建议继续投，但打法必须从“堆量”切到“结构优化”。**

理由有三点：
- 燕云不是没有需求。短视频总 VV 已到 4.05 亿，直播总看播时长也有 1161 万分钟，说明平台内已经形成稳定内容消费。[^1][^2]
- 真正短板不在需求侧，而在供给结构。短视频供给已高于竞品中位数，但效率只有中位数的 64%；直播单位开播产出接近中位，但主播规模只有中位的 48%。[^1][^2]
- 历史复盘也验证了燕云在 TikTok 上具备被预算放大的能力：GIP 短视频对投稿和自然 VV 有显著贡献，直播节点战役也确实能拉高看播规模。[^4][^5]

#### 6.2 投短视频还是投直播
**优先级上，先稳短视频提效，再做直播补量。**

- **短视频** ：是燕云当前更强的用户触达面，也是最明显暴露结构问题的模块。后续预算更适合投给 **高质量创作者筛选、跨垂类内容模板、重点国家定向加码** ，而不是继续让低效供给自然堆高。[^1][^4]
- **直播** ：仍值得投，但应当以 **重点国家 + 重点主播** 为主，优先补足高承接市场的供给缺口，而不是把预算撒到所有区域。[^2][^4]

#### 6.3 投哪些国家 / 哪些达人 / 哪些主播
**国家建议**
- **短视频优先** ：DE、VN、US、ID。
- **直播优先** ：VN、US、ID。
- **谨慎控制** ：MY、FR、KR 的短视频铺量；PH、TH 的直播粗放补量。[^1][^2]

**达人建议**
- 短视频优先选 **能做剧情解释、文化解读、角色演绎、开放世界体验表达** 的作者，不建议继续大量吃泛游戏低质铺量。[^4]
- 继续放大已经被验证的 **跨垂类作者合作** ，因为燕云的内容优势不只在玩法，还在“文化细糠”“剧情沉浸感”“武侠审美”这些更容易破圈的表达。[^4]

**主播建议**
- 直播优先扶持 **头部 + 强中腰部** 、且已有稳定看播承接的主播池。
- 不建议继续把预算大量给低承接长尾主播；历史活动复盘已经证明，燕云直播的有效打法更接近 **强头部带动 + 中腰部铺量** ，而不是纯长尾扩张。[^4]

### 7. 对项目方可直接复用的话术
燕云当前在 TikTok 上不是没有需求，问题主要不在消费者不看，而在内容供给还没有完全长成第一梯队。短视频侧已经把量做起来了，但条均vv还没进入头部竞品水平，说明后续预算更应该用来筛达人、提内容效率，而不是继续平均铺量。直播侧的情况更积极一些，单位开播产出已经接近竞品中位，说明主播只要补到对的国家、对的人群，还是有继续放大的空间。接下来更建议把预算优先投到 DE / VN / US / ID 的短视频，以及 VN / US / ID 的直播，用高质量创作者和高承接主播去放大燕云本身的内容优势，而不是在低承接市场重复堆量。[^1][^2][^4][^5]
